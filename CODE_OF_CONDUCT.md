# Code Of Conduct

## Our Standard

This project aims to be welcoming, respectful, and constructive.

Participants are expected to:

- be respectful in discussions and reviews
- assume good intent while still giving direct, helpful feedback
- focus criticism on ideas, code, and behavior rather than people
- make space for contributors with different experience levels

Unacceptable behavior includes:

- harassment, intimidation, or personal attacks
- discriminatory language or exclusionary behavior
- publishing private information without permission
- trolling, derailing, or repeatedly ignoring reasonable project boundaries

## Enforcement

Maintainers may edit, remove, or reject comments, issues, pull requests, and other contributions that do not align with this code of conduct.

## Reporting

If someone is making the project feel unsafe or hostile, contact the maintainer privately. Please include links, screenshots, or other context if possible.
